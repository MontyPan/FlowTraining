這是我理解 git 是什麼東西
-------------------------



git 是用於版本控制的工具，優點是
*	分散式開發 
	*	即使沒有伺服器或者網路的環境，依舊可以使用 git 進行版本控制。
*	速度快
	*	由於是完整複製一份在本機，包含完整檔案及記錄檔，所以針對版本控管中的各種檔案操作速度，將會比直接從遠端存取來的快上百倍之多。
*	免費
	*	免費就是讚
