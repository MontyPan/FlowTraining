# 我在這個 task 中理解到的 git 是什麼東西

* Git 是一種分散式版本的版本控制系統
* Git 可以清楚的記錄每個檔案是誰在什麼時候加進來、什麼時候被修改或刪除
* Git 幫你保留這些歷史紀錄跟證據，萬一出事的時候你就能知道是從什麼時候開始就有問題，以及知道該找誰負責，再也不用自己背黑鍋


如需更進一步了解 git 內容，可參考[為你自己學 Git](https://gitbook.tw/)
